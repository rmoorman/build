Introduction
============

* This is a buildout you can use to install cyn.in (http://www.cynapse.com/cynin)

Install
=======

* To install Cyn.in follow these steps:

    % svn export https://svn.aclark.net/svn/public/buildout/cynin/trunk/
    % python2.4 bootstrap.py
    % bin/buildout
    % bin/instance fg

* Open http://localhost:8080/manage

* Login with admin:admin.

* Add a Plone site called Plone

* In the portal_quickinstaller in the ZMI, install Scrawl, then Ubi*

* Open http://localhost:8080/Plone

Have fun!
