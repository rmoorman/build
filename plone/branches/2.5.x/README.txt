Introduction
============

This is a Plone 2.5.x buildout.

Installation
------------

    $ virtualenv-2.4 plone
    $ cd plone
    $ bin/easy_install zc.buildout
    $ curl http://dist.aclark.net/build/plone/2.5.x/extends.cfg > buildout.cfg
    $ bin/buildout
    $ bin/instance fg
