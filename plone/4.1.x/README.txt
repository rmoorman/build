Introduction
============

This is a Plone 4.1.x buildout.

Installation
------------

    $ virtualenv-2.6 plone
    $ cd plone
    $ bin/easy_install zc.buildout
    $ curl http://dist.aclark.net/build/plone/4.1.x/extends.cfg > buildout.cfg
    $ bin/buildout
    $ bin/plone fg
